## 项目概述
本项目是为第 16 届上海交通大学新生机械创意大赛设计的**麦卡纳姆轮全向移动机器人**。
已完成**Xbox蓝牙手柄控制、MPU6050智能坡道自动驻车、4路舵机机械臂控制、步进电机升降、电池低压保护、编码器速度滤波**全套核心功能，完全满足初赛/决赛关卡需求。

## 控制系统架构

### Main_Control - 底盘+机械臂一体化控制（最终正式版）
**硬件**：ESP32-S3-DevKitC-1 + I2C电机驱动(0x26) + I2C舵机驱动(0x2D) + MPU6050

#### 控制映射（与最新代码完全同步）
| 手柄输入 | 机器人动作 |
|---------|-----------|
| 左摇杆↑↓ | 底盘前后移动 |
| 左摇杆←→ | 底盘左右平移 |
| 右摇杆←→ | 底盘原地旋转 |
| 右摇杆↑↓ | 升降平台控制 |
| Share/Start/Select | 舵机一键复位（3组预设） |
| A/B | 舵机4（夹爪）开关 |
| X/Y | 舵机3（臂身）调节 |
| 十字键 | 舵机1/2（基座/大臂）调节 |
| RT/RB | 机械臂纵深协调控制 |

#### 核心参数（最新校准版）
```cpp
PWM_MAX = 3600        // 最大输出
PWM_MIN = 50          // 最小启动
PWM_DEADZONE = 2000   // 摇杆死区
SLOPE_COMPENSATION_STEPS = 350  // 坡道驻车力度
SPEED_CURVE_EXP = 1.5  // 速度曲线指数
ANGLE_THRESHOLD = 12.0° // 驻车触发倾角
```

## 快速开始
### 环境搭建
1. 安装 VSCode + PlatformIO 插件
2. 打开 `Main_Control` 项目
3. 核对手柄MAC地址：`#define XBOX_MAC "2d:89:6d:41:41:6a"`
4. 编译上传：
```bash
pio run --target upload
pio device monitor
```
